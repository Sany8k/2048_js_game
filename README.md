# JavaScript 2048 game

2048 JS Game is a web-based implementation of the classic sliding tile puzzle.

It features complete core game mechanics, including dynamic tile generation, smooth merging, real-time score tracking, and win/lose state detection. Built with pure JavaScript, HTML, and CSS/SCSS, the project ensures lightweight performance and cross-device compatibility without relying on external libraries or frameworks.

## Getting Started

1. Clone the repository:
   
```bash
git clone https://github.com/Sany8k/2048_js_game.git
cd 2048_js_game
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the project locally:

```bash
npm start
# or
yarn start
```
